# HRSAAS

人力资源管理中台

> 项目基于 vue3-admin-template [Github](https://github.com/jzfai/vue3-admin-template)

### views

```md
├── approvals # 审批/批准
├── attendances # 考勤（/əˈtendəns/）
├── dashboard # 首页/面板/仪表盘
├── departments # 组织架构/部门
├── employees # 员工
├── error-page # 错误页（404，401）
├── login # 登录
├── permission # 权限管理
├── salarys # 薪水/工资
├── setting # 公司设置
├── social # 社保/社会
```
